HOW TO USE MAPFORGE

This is the map of the whole game world. Every click saves instantly into game/data/usmap.json — the exact file the game loads. There is no Save button because you never need one; the amber flash in the bottom-right corner confirms each write. Restart the game and your world is live.

The workflow story

Build the world in this order:

  1. Arrange the roads — the interstates are the spine of everything.
  2. Drop EXIT nodes along them — each exit is a socket where content will live.
  3. (Later) place communities — towns and structures grow at those exits.

In one line: arrange roads → drop exits → (later) place communities.

The tools (top of the left bar — the lit one is active)

PAINT — paints the land itself. Pick a color in the Biome palette (hover one to read what it grows in-game), pick a Brush size (1/3/5/9 cells — the number keys work too), then drag across the map. The stroke saves the moment you release the mouse.

ROAD — draws and extends roads. First pick a road in the Roads box — it lights up bright amber, everything else goes dim. Then every click on the map drops that road's next point; the numbered dots show the order and the red dot is the newest. UNDO PT removes the last point, NEW starts a fresh road, DELETE removes the whole selected road. Before you hit NEW, set kind (interstate / county / street / dirt), lanes, surface and divided — those build the road you're about to draw, and picking a kind proposes the rest. Then name it (like US-50) and click 2+ points. Editing points never touches the rest of a road's data — nicknames (THE CRIMSON MILE), danger, families, tolls, lanes, surface and the ramp/interchange markings a bake wrote all survive your edits.

TOWN — founds a town. Click the map, type a name, done — it appears as a named dot.

PLACE — pins ONE structure at the exact spot you click. Choose which one under Authored structures first. The land around it stays procedural; only that building is guaranteed.

EXIT — the important one. Click where the exit should LEAD — the destination, not the highway itself. A little dialog opens:

  1. Name it, or leave the name blank and it names itself after the archetype.
  2. Pick an archetype — what kind of stop this is; each carries a default community tier and a danger level.
  3. Keep return ramp checked for a full interchange (a way back onto the highway).
  4. Press BUILD EXIT.

The node anchors itself on the nearest interstate automatically, and in the game the off-ramp, the return ramp and the big highway sign all build themselves. The Exit nodes list shows every node you've made; its ✕ deletes a node and its ramp roads with it.

INSPECT — click any cell and the bottom bar reads out its biome, its state, and the nearest road and town with distances in meters.

Zoom & layers

− / + zoom (the mouse wheel over the map works too), FIT snaps back to 100%. Scroll to pan when zoomed in. STATE LINES / ROADS / TOWNS only show or hide those layers on screen — they never change the data.

Exits & templates (the older tools)

AUTO-EXIT is the v1 ramp builder: type a town id and it lays a bare ramp to that town. The EXIT tool above does everything this does and more — prefer the EXIT tool. STAMP drops the chosen template (waystation / hamlet / outpost) of structures at your last click on the map — so click the spot first, then press STAMP and name the place.

The structure catalog

Click ▸ Structure catalog to unfold the library of building types. A structure profile is the blueprint of a kind of building — its name, sign glyph, size, floors, danger, doors, loot and workers — not a building standing in the world. Created ≠ placed: saving a profile only puts it on the shelf; it enters the world later, when the PLACE tool, a template, or the community generator uses it.

NEW clears the form for a fresh profile, SAVE writes it into the catalog, DELETE removes it.

The §9 “every building needs a job” rule: if SAVE refuses with a red NOT LAWFUL message naming §9, the profile is missing its reason to exist. Give it at least ONE of: a loot table (something to find), npc jobs (someone works there), law hooks (a rule applies there) or event hooks (something can happen there). A building with none of those is dead scenery — and the catalog refuses dead scenery.

Housekeeping

RE-READ FROM DISK reloads the map file — use it if the game, an AI, or another tab changed the map underneath you. The footer's left side always shows the cell, world coordinates and biome under your cursor.

Keys: 1/3/5/9 brush size · Esc closes dialogs and this guide. Hover any button for a one-line reminder.
