# music/game/ — game score & ambient beds

**Drop here:** `.mp3` tracks for the game itself (not the diegetic radio):
menu themes, safehouse ambience, moment stingers, drone beds.

**What MediaForge does:** lists what's here in its scan panel
(http://localhost:8897). Its GENERATE TEST MUSIC button drops
`test_ambient.mp3` here to prove the pipeline.

Game code picks from this folder by name/prefix — keep names descriptive
(`menu_theme.mp3`, `safehouse_night.mp3`).
