# music/radio/ — radio station music

**Drop here:** `.mp3` tracks for the in-game radio's MUSIC stations (the Y-scan
dial). The folder IS the playlist — no conversion, no manifest row needed.

**What MediaForge does:** lists what's here in its scan panel
(http://localhost:8897) so you can see the station's library at a glance. Its
GENERATE TEST MUSIC button drops `test_wasteland_loop.mp3` here to prove the
pipeline.

Keep tracks loudness-consistent; the radio plays them as-is.
