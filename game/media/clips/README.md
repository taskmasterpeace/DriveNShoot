# clips/ ‚Äî short ambient inserts

**Drop here:** short pieces as `.mp4` / `.mov` / `.mkv` / `.webm`: news
segments, bumpers, loops, fake commercials, propaganda, scene excerpts.
Anything a public screen can run unattended.

**What MediaForge does:** encodes each source to `clips/<id>/<id>.ogv`
(Theora/Vorbis), extracts `poster.png`, probes runtime, writes a
`category: "clips"` row to `game/data/media_manifest.json`.

The synthetic **DRIVN TEST REEL** (`test_pattern/`) is generated by
MediaForge's GENERATE TEST CLIP button ‚Äî it proves the in-game TV pipeline
before any real footage is dropped. Safe to delete once real clips exist
(delete the row in the UI too).
