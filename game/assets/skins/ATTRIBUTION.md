# Pixel-art world skins

16×16 seamless wasteland material tiles for the "pixel art in 3D" look (NEAREST-filtered,
texel-per-meter, via ProtoWorldBuilder.material_textured / material_skin).

**Source:** generated with [PixelLab](https://pixellab.ai) (tiles-pro, square_topdown,
segmentation, seed 70707) — road/wall/dirt/metal/wood/rust, wasteland amber/bone/rust palette.
Procedural fallbacks live in tests/skin_gen.gd (same PNG names) if a skin is ever missing.
